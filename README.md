Бот для занятий спортом

Бот, исходя из данных о пользователе, генерирует план тренировок и предоставляет калькуляторы, для помощи в поддержании здоровья.

Задача 1.

Создание заготовки чат-бота для Telegram, которая умеет отвечать на несколько команд:

1) /about — выводит краткую информацию про бота
2) /authors — выводит информацию об авторах
3)/help — выводит информацию обо всех доступных командах (можно запускать с аргументом: /help <command>)

Задача 2.

Создан интерфейс для работы с базой данных

1) Создана база данных
2) Создан класс-менеджер для работы с базой данных
3) добавлены inlineKeyboard для упрощенного ввода данных
4) реализованы команды /start, /profile, /profile edit

Задача 3.

созданы калькуляторы

создана команда /calculators, которая выводит несколько калькуляторов на выбор(inline keyboard): подсчет калорий(с учетом цели пользователя), расчет бжу, калькулятор имт, расчет нормы воды.
